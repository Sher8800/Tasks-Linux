1. Прописываем следующую команду: #!/bin/bash;
    (данная строка дает системе понять что скрипт нужно запускать при помощи bash, который находится в папке /bin)
  2. Далее прописываем следующие команды:
    for run in {1..10} -
    do
    echo $run
    sleep 1
    done

В каждой итерации цикла в переменную run команда echo будет выводить следующее значение из фигурных скобок ”списка”. В первом цикле выведет первое значение, во втором — второе, и тд. пока цикл не дойдёт до последнего указанного нами значения.
